Zoom's Real-Time Media Streams (RTMS) is a WebSocket-based streaming API delivering live access to meeting audio, video, transcripts, and participant events without requiring bot participants. Now in General Availability via Zoom's Developer Pack or Recall.ai, it provides raw audio packets, video as JPG/PNG/H.264, and live transcripts, but is Zoom-only, host-dependent, receive-only, lacks breakout room support, and requires Zoom client 6.5.5 or later.
